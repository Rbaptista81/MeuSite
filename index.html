Introdução

Como criar um site simples em HTML e publicá-lo na web utilizando o GitHub Pages. O objetivo é aplicar os conhecimentos de HTML e CSS. Desafio: o meu Website online!

Estrutura

O site é dividido em seções com navegação por âncoras. A imagem de fundo tem um visual tecnológico, e a estrutura é composta por HTML sem frameworks externos.

Código

Todo o código está em um único ficheiro HTML.

Publicação

O site é publicado através do GitHub Pages.

Conclusão

Este é o meu trabalho, usando HTML e CSS.
