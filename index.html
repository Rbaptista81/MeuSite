1. Início e Objetivo

O objetivo deste site é demonstrar como construir e publicar uma página simples na web utilizando HTML e GitHub Pages.

2. Estrutura do Site

Um site básico é composto por arquivos HTML, CSS e, eventualmente, imagens ou JavaScript. Aqui usamos apenas HTML e CSS.

3. Código HTML

O código da página é estruturado com um cabeçalho, uma navegação simples com âncoras e conteúdo dividido em seções.

4. Publicação com GitHub Pages

Após criar o repositório no GitHub e fazer o upload do ficheiro index.html, ativamos o GitHub Pages nas configurações.
